A horizontal line element used to visually separate groups of menu items within a ContextMenu.

## Constructor

```csharp
new MenuDivider();
```

## Examples

### Grouping Menu Items
Use dividers to organize related menu items into logical groups:

```csharp
new ContextMenu(
    // Edit group
    new MenuItem("Cut", () => Cut()),
    new MenuItem("Copy", () => Copy()),
    new MenuItem("Paste", () => Paste()),

    new MenuDivider(),

    // Selection group
    new MenuItem("Select All", () => SelectAll()),
    new MenuItem("Deselect", () => Deselect()),

    new MenuDivider(),

    // Danger zone
    new MenuItem("Delete", () => Delete())
);
```

### Conditional Dividers
Show or hide dividers based on application state:

```csharp
var HasRecentFiles = new ReactiveProperty<bool>(true);

new ContextMenu(
    new MenuItem("Save", () => Save()),
    new MenuItem("Save As...", () => SaveAs()),

    new MenuDivider()
        .BindVisible(HasRecentFiles),

    // Recent files section only shown when there are recent files
    new MenuItem("Recent File 1", () => OpenRecent(0))
        .BindVisible(HasRecentFiles),
    new MenuItem("Recent File 2", () => OpenRecent(1))
        .BindVisible(HasRecentFiles)
);
```
