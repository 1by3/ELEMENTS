A clickable menu item element for use within ContextMenu. Supports optional icons and automatically closes the parent menu when clicked.

## Constructor

```csharp
// Text only
new MenuItem("Edit", () => Edit());

// With icon from Resources path
new MenuItem("Icons/edit", "Edit", () => Edit());

// With Texture2D icon
new MenuItem(editIcon, "Edit", () => Edit());
```

## Examples

### Basic Menu Item
Simple text menu items with click handlers:

```csharp
new ContextMenu(portal,
    new MenuItem("Cut", () => Cut()),
    new MenuItem("Copy", () => Copy()),
    new MenuItem("Paste", () => Paste())
);
```

### Menu Items with Icons
Add icons to menu items for better visual recognition:

```csharp
new ContextMenu(portal,
    new MenuItem("Icons/cut", "Cut", () => Cut()),
    new MenuItem("Icons/copy", "Copy", () => Copy()),
    new MenuItem("Icons/paste", "Paste", () => Paste())
);

// Or using Texture2D
new MenuItem(myTexture, "Custom Action", () => DoAction());

// Or fluently
new MenuItem("Edit", () => Edit())
    .Icon("Icons/edit");
```

### Disabled Menu Items
Disable menu items based on application state:

```csharp
new ContextMenu(portal,
    new MenuItem("Cut", () => Cut())
        .Disabled(!hasSelection),
    new MenuItem("Copy", () => Copy())
        .Disabled(!hasSelection),
    new MenuItem("Paste", () => Paste())
        .Disabled(!hasClipboardContent)
);

// Or with reactive binding
new MenuItem("Delete", () => Delete())
    .BindDisabled(ViewModel.IsLocked);
```

### Dynamic Label
Update menu item labels dynamically:

```csharp
var menuItem = new MenuItem("Undo", () => Undo());

// Later, update the label
menuItem.Label("Undo (Ctrl+Z)");
```

## Behavior

- **Auto-close**: When a MenuItem is clicked, it automatically closes its parent ContextMenu
- **Disabled state**: Disabled menu items do not trigger click handlers and show reduced opacity
- **Icon positioning**: Icons are displayed to the left of the label text

## CSS Classes

- `.elements-menu-item` - The menu item container
- `.elements-menu-item:hover` - Hover state styling
- `.elements-menu-item.unity-disabled` - Disabled state styling

MenuItem extends [BaseElement](/docs/elements/base-element).
