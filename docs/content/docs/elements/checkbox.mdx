A toggle input element for boolean values. Wraps Unity's Toggle with shadcn-inspired styling â€” a clean bordered box that fills dark with a white checkmark when checked.

## Constructor

```csharp
new Checkbox();
new Checkbox(reactiveProperty); // Bound to ReactiveProperty<bool>
```

## Examples

### Two-Way ReactiveProperty Binding
Checkboxes provide automatic two-way binding with ReactiveProperty:

```csharp
// Constructor binding (preferred)
new Checkbox(AcceptTerms)
    .Label("I accept the terms and conditions");

// Explicit binding
new Checkbox()
    .BindValue(AcceptTerms)
    .Label("I accept the terms and conditions");

// When user toggles, AcceptTerms.Value updates automatically
// When AcceptTerms.Value changes, the checkbox updates
```

### Form Fields
Common form field patterns:

```csharp
// Simple toggle
new Checkbox(RememberMe)
    .Label("Remember me");

// Disabled checkbox
new Checkbox(LockedOption)
    .Label("Premium feature")
    .Disabled(true);

// Checkbox without label
new Checkbox(IsEnabled);
```

### Reactive State Management
Checkboxes can respond to state:

```csharp
// Conditionally disabled based on other state
new Checkbox(AcceptTerms)
    .Label("I accept the terms")
    .BindDisabled(IsSubmitting);

// Dynamic label
new Checkbox(DarkMode)
    .BindLabel(DarkMode.Select(on =>
        on ? "Dark mode (on)" : "Dark mode (off)"));
```

### Change Callbacks
React to value changes without binding:

```csharp
new Checkbox()
    .Label("Enable notifications")
    .OnChange(enabled => Debug.Log($"Notifications: {enabled}"));
```
